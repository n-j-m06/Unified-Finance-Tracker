# 💰 Unified Finance Tracker
**Full-Stack Personal Finance Management System**

A comprehensive financial tracking solution designed to help users monitor income, manage expenses, and gain insights into their spending habits. This project features a robust Python backend and a modern, interactive React frontend to provide a seamless data-driven experience.

---

## 🚀 Key Features
*   **Income & Expense Management:** Real-time tracking of financial inflows and outflows with detailed categorization.
*   **Dynamic Visualizations:** Interactive displays for monthly income and expenditure trends.
*   **Secure Authentication:** User-specific dashboards and data persistence.
*   **Automated Calculations:** Precise financial summaries and balance tracking to ensure data integrity.
*   **Responsive Design:** Optimized for various screen sizes using modern CSS frameworks.

## 🛠️ Tech Stack
*   **Frontend:** React.js, Tailwind CSS, Lucide-react
*   **Backend:** Python (FastAPI/Flask), SQL (PostgreSQL/Neon)
*   **State Management:** React Hooks (useState, useEffect)
*   **Styling & Animation:** Tailwind CSS, Framer Motion
